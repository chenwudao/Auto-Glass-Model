# Auto-Glass 项目说明

## 项目简介

Auto-Glass 是一个专注于眼部健康监测与预测的智能系统，主要通过机器学习和计算机视觉技术，分析和预测不同年龄段人群的眼轴长度变化，以及其他眼部相关指标。该项目旨在帮助用户了解影响眼部健康的因素，及时发现潜在的视力问题。

## 主要功能模块

### 1. 眼轴长度预测模块 (axiallength1)

该模块主要功能是预测不同年龄组在一天中不同时间段的眼轴长度变化。

- **核心功能**：
  - 基于年龄组和时间预测眼轴长度
  - 生成不同年龄组的眼轴长度日变化图表
  - 提供用户输入预测界面

- **技术实现**：
  - 使用随机森林回归模型进行预测
  - 利用LabelEncoder对年龄组进行编码
  - 生成模拟数据并进行可视化

- **使用方法**：
  ```python
  # 运行预测和可视化
  python axiallength1/function.py
  ```

### 2. 面部检测与实时预测模块 (facialdetect3)

该模块实现了基于面部检测的实时眼轴长度和晶状体厚度预测系统。

- **核心功能**：
  - 实时面部检测和定位
  - 提取面部和眼部特征
  - 预测眼轴长度(AL)和晶状体厚度(LT)
  - 实时显示预测结果和相关特征

- **技术实现**：
  - 使用SVR模型进行预测
  - 模拟YOLO人脸检测
  - 实时视频流处理
  - 模拟环境数据和面部特征提取

- **使用方法**：
  ```python
  # 运行实时预测系统
  python facialdetect3/facedetect.py
  ```

### 3. 影响因素分析模块 (factors2)

该模块分析影响眼轴长度变化的各种因素，并建立预测模型。

- **核心功能**：
  - 分析光照强度、瞳孔直径、环境评分和认知负荷对眼轴长度的影响
  - 建立SVR预测模型
  - 分析特征重要性

- **技术实现**：
  - 使用SVR模型进行预测
  - 使用SHAP进行特征重要性分析
  - 数据标准化和预处理

- **使用方法**：
  ```python
  # 运行影响因素分析
  python factors2/svmmodel.py
  ```

## 技术栈

- **编程语言**：Python
- **机器学习库**：scikit-learn, shap
- **数据处理**：pandas, numpy
- **可视化**：matplotlib
- **计算机视觉**：OpenCV, YOLOv8
- **其他**：随机森林回归, 支持向量回归(SVR)

## 项目亮点

1. **多维度分析**：从年龄、时间、环境、面部特征等多个维度分析眼部健康
2. **实时预测**：结合计算机视觉技术实现实时眼部指标预测
3. **可视化展示**：生成直观的眼轴长度变化图表
4. **特征重要性分析**：使用SHAP分析影响眼轴长度的关键因素
5. **用户友好**：提供简单的用户输入界面进行预测

## 应用场景

1. **眼部健康监测**：实时监测眼部指标，及时发现异常
2. **视力保护指导**：根据环境和使用习惯提供视力保护建议
3. **研究分析**：为眼部健康研究提供数据支持和分析工具
4. **教育领域**：监测学生用眼情况，预防近视

## 安装与运行

1. **克隆项目**：
   ```bash
   git clone <项目地址>
   cd Auto_Glass
   ```

2. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```

3. **运行各模块**：
   - 眼轴长度预测：`python axiallength1/function.py`
   - 实时面部检测：`python facialdetect3/facedetect.py`
   - 影响因素分析：`python factors2/svmmodel.py`

## 注意事项

- 本项目使用的是模拟数据，实际应用中需要使用真实数据集进行模型训练和验证
- 面部检测模块需要摄像头支持，运行时请确保摄像头已连接且未被其他程序占用
- 部分功能可能需要较高的计算资源，建议在配置较好的设备上运行
